AWS REST API

I created a REST API using AWS API Gateway backed by an AWS Lambda function. This API essentially acted as a wrapper for an existing SOAP based web service allowing for easier calls from a mobile platform. In my AWS Lambda function I put some Java JAX-WS client code so that I was able to call the SOAP based service. One of the biggest challenges of this project was developing the API within a corporate environment (which forced me to use the correct VPC, IAM roles, etc. within AWS).

Our team utilized Rally Software during development to keep track of current tasks and project progress.
